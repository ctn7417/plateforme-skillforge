# Plateforme Skillforge

## Flux utilisateur minimal

1. **Inscription (signup)** : création de compte et connexion.
2. **Parcours unique** : accès à un seul parcours publié et actif.
3. **Questionnaire unique** : passage d'un seul quiz rattaché au parcours.
4. **Certificat** : génération du certificat après validation des critères.

## Fonctionnalités retirées (non critiques)

- Social (amis, commentaires, fil d’actualité).
- Badges et gamification secondaires.
- Analytics avancés (dashboards détaillés, cohortes, funnels personnalisés).

## Règle d’éligibilité au certificat

- Certificat délivré si le **score du quiz ≥ 70 %**.

## Validation du flux minimal

- Le flux doit fonctionner avec **un seul parcours** actif.
- Chaque parcours n’a **qu’un seul questionnaire** associé.
- Le certificat est **généré uniquement** après le passage de ce questionnaire.
